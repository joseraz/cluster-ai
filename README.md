# Welcome to Cluster AI

## Project info

Visualize your network of contacts to strengthen your connections.

## Development

```bash
npm run dev              # Vite UI + Hono API
npm run storybook        # Isolated UI development harness
npm run build-storybook  # Static Storybook validation build
npm run test             # Vitest API and source-level tests
```
